# PRASHNA — KP Prashna Astrology

A self-contained, offline horary (Prashna) astrology web application based on **Hora Shastra** and **Krishnamurti Paddhati (KP)**.

## Features
- Manual Prashna (enter date, time, place) and Instant Prashna (uses current time + your location)
- In-browser computation: sidereal planetary positions (KP ayanamsa), Placidus cusps, nakshatra star-lords and sub-lords
- KP 1–249 horary number, house significators, and Ruling Planets used to judge fructification
- Elaborate dual-system answers (KP + Hora Shastra) for all 12 houses: health, marriage, career, litigation, travel, children, finance, property, education, enemies, longevity, and missing/lost articles
- Detailed missing-item analysis: old/new, material, misplaced vs lost vs stolen, location/direction/landmarks, thief profile and recovery prospects

## Usage
Open `index.html` (or `kp-prashna.html`) in any modern web browser. No installation, server, or internet connection is required.
